Oque é oque é?

PSEUDOCODIGO?

Pseudocódigo é uma representação simplificada e informal de um algoritmo, escrita numa linguagem natural (como português ou inglês) que se assemelha ao código de uma linguagem de programação, mas que não é executável. pseudo codigo

HTML?

HTML é a sigla para HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto.

CSS3?

CSS3 é a terceira versão da linguagem Cascading Style Sheets (CSS), responsável por definir a aparência e o layout de páginas web, introduzindo novas funcionalidades e módulos que permitem maior flexibilidade e criatividade no design.
